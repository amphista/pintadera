Pintadera

A fast, ad-free click-to-copy picker for 8,000+ Unicode symbols, glyphs & emoji.

Click any glyph to copy it — then paste it wherever you need. Search 8,000+ Unicode symbols, punctuation, math, arrows, technical keys and emoji, and preview how each one actually renders per platform before you use it. Useful anywhere you type — and it can build a Stream Deck “Symbols” profile too.

⌕

Building a Stream Deck Symbols profile? Turn on Select, tap the glyphs you want, then Export. You’ll get a ready-to-paste prompt for Claude Code that uses the Stream Deck MCP to create the profile — folders, pages, and a System ▸ Text action on every key that types the symbol. The export reserves the two bottom corners of each XL page for Back + Next, so 30 glyphs map to one physical page.

About. Character data is generated from the Unicode Character Database; emoji metadata and the Apple / Google / Twitter preview images come from emoji-datasource via jsDelivr; the cross-platform line-art reference uses Google’s Noto symbol fonts. CJK, Hangul and other non-Latin scripts are intentionally excluded. Free & open — no tracking, no backend.

Pintadera is a Two Actual Eyes tool — no ads, no nonsense. If it saved you a search, you can buy me a coffee.
