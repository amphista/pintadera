Symbols Profile

Tap a folder, page through the keys, click any glyph to copy it, then paste into the Stream Deck button's Text action. Pages are sized to 30 keys so one page here maps to one folder page on the XL (2 keys left for Back + Next).

XL · 8×4
Setup, fast version. For each key: drop the System ▸ Text action on it, paste the symbol into the text field, and set the key title (or icon) to the same glyph so you can read it at a glance. Reserve the bottom-left key of every folder for Create Folder ▸ back and the bottom-right for the next page. Emoji and some music/astro glyphs are color or multi-codepoint; they type fine but may show monochrome on the device face, so a clear title helps.
Note. The full Unicode set is ~150k characters, so this is the genuinely useful slice for design, dev, writing, and your chart work, not every codepoint in existence.
